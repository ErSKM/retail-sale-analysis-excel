# Retail Sales Performance Analysis (Excel Project)

## Project Overview
This project analyzes retail sales data using Microsoft Excel to understand sales trends, regional performance, product category contribution, and overall profitability.

## Dataset
The dataset consists of multiple tables:

- Transactions: 5000 records
- Customers: 200 records
- Products: 50 records
- Stores: 5 records

These tables were combined to create a master dataset for analysis.

## Data Preparation
The following steps were performed:

- Combined multiple tables using XLOOKUP
- Created a Sales Master Data table
- Generated calculated columns:
  - Revenue
  - Profit
  - Month
  - Year

## Analysis Performed

### Sales by Region
Identified regional performance across stores.

### Sales by Category
Analyzed which product categories generate the most revenue.

### Monthly Sales Trend
Studied seasonal sales patterns.

### Top Products
Identified products contributing most to revenue.

## Dashboard Features
- KPI metrics
- Sales trend charts
- Category performance visualization
- Interactive slicers

## Tools Used
Microsoft Excel  
Pivot Tables  
Data Visualization

## Key Insights
- East region generated the highest revenue.
- Electronics category contributed the largest share of sales.
- Sales fluctuated across months indicating seasonal demand.
- Profit margin remained strong at around 35%.
